# Compilation et éxecution

La compilation nécessite un compilateur C++14

## Windows
- Lancer gkit2light.sln dans build

## Linux 

- Installer gkit2light et suivre la procédure [https://perso.univ-lyon1.fr/jean-claude.iehl/Public/educ/M1IMAGE/html/group__installation.html](url)

- Build avec premake4 en utilisant premake4.lua
